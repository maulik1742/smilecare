## About the Project

**ClinicSite** is a professional, lightweight website built specifically for **medical clinics, doctors, dentists, and healthcare providers**. The goal is simple — help clinics establish a strong online presence, build patient trust, and make it easy for new patients to find and contact the clinic.

This project was built as part of a freelance portfolio to demonstrate the ability to design and deliver real-world, production-ready websites for local healthcare businesses.

Whether you are a **general physician, specialist, dental clinic, or diagnostic center** — this template is built for you.

## Key Features

- Lightning Fast — Optimized for Core Web Vitals and Google PageSpeed
- Fully Responsive — Pixel-perfect on all screen sizes
- Clean UI/UX Design — Professional look that builds patient trust instantly
- SEO Optimized — Meta tags, semantic HTML, and structured data ready
- Click-to-Call Button — One tap to call the clinic directly from mobile
- Google Maps Integration — Patients can locate the clinic with ease
- Working Hours Display — Clear clinic timings shown for patients
- WhatsApp Chat Button — Direct WhatsApp inquiry from the website
- Services Showcase — Display all treatments and specialties offered
- Doctor Profile Section — Build credibility with qualifications and experience
- Patient Testimonials — Social proof to convert visitors into patients
- Appointment CTA — Strong call-to-action to book appointments

## Getting Started

### Prerequisites

- A web browser
- A code editor (VS Code recommended)
- Git installed

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/clinic-website.git

# 2. Navigate into the project folder
cd clinic-website

# 3. Open in your browser
open index.html
```

### Deploy to Vercel

```bash
# Install Vercel CLI
npm install -g vercel

# Deploy
vercel

# Done! Your site is live in under 60 seconds.
```
